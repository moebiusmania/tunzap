1. Iniziamo: le basi e qualche Css

Doctype (solo uno)

Il Doctype e' quella stringa che indica al browser quale versione di Html stai usando.

  • Html 5
  • Html 4 - Transitional
  • Html 4 - Strict

Reset/Framework CSS (anche piu' di uno)

I framework e i reset Css possono sensibilmente velocizzare la realizzazione di layout web, se ci prendi un po' la mano...

  • YUI3 CSS Reset
  • 960 Grid System
  • Twitter Bootstrap
  • 1140 CSS Grid

2. Scegli le librerie Javascript che ti servono per il tuo nuovo progetto.

Framework Javascript (scelta multipla)

E' consigliabile se possibile lavorare con una sola libreria Javascript,ma se il progetto lo richiede qui ci sono le 4 piu' diffuse.

  • jQuery
  • MooTools
  • Prototype
  • Ext Core

Utility (scelta multipla)

Altre librerie ti permettono di aggiungere interattivita' evolute come il drag'n'drop, o l'inserimento semplificato di file Swf.

  • jQuery UI (richiede jQuery!)
  • HTML5 Shiv
  • SWF Object
  • Script.aculo.us (richiede Prototype)

3. Cartelle necessarie, layout di partenza... e via!

Cartelle di progetto (scelta multipla)
